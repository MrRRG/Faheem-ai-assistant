# FAHEEM – AI Government Digital Assistant

FAHEEM is a graduation project developed as part of the Information Systems program at King Faisal University.

The goal of the project is to build an AI-powered assistant that helps users access government services more easily and efficiently.

Features:
- Intelligent assistant for government services
- User-friendly interface
- Support for digital service guidance
- AI-based interaction concept

Technologies / Concepts:
- Artificial Intelligence
- Information Systems
- System Analysis
- Web-based services
